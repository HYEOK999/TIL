## Prototype

### 캡슐화 ( 정보 은폐 )

다른 클래스 기반 프로그램언어에서 `public` , `private`, `protected` 를 이용해서 캡슐화를 하여 클래스를 보호하면 정보를 은폐한다.

하지만 JS에서는 **클로저**를 통해 캡슐화를 지원한다. (추후 버전에서는 JS도 클래스를 출시 한다.)
